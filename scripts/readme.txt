    # 주요 기능

파일명 기반 자동 분류: 카테고리_SNS_analysis.csv 형식의 파일명을 해석하여 데이터의 속성(Category, SNS)을 자동으로 부여합니다.

데이터 동기화 (Sync): 동일한 카테고리와 SNS의 데이터가 이미 존재할 경우, 기존 데이터를 삭제하고 최신 데이터로 교체합니다.
+DB 저장 시간대 기점으로 최신 3회차까지의 데이터를 보존하고, 초과되면 가장 오래된 회차를 삭제하는 기능을 추가했습니다.
+기준일로부터 30일 전까지의 데이터를 보존하게끔 했고, 빈도수 3미만의 데이터는 제외하고 저장하는 것으로 수정했습니다.

벡터화 (Vectorization): Upstage 임베딩 모델을 사용하여 텍스트 데이터를 수치화하고 검색 가능한 상태로 저장합니다.


    # 사용 준비

전처리된 CSV 파일을 scripts/ 폴더에 위치시킵니다. 파일명 규칙을 반드시 준수해야 합니다.

규칙: {카테고리}_{SNS}_20aaddbb.csv

예시: food_youtube_20260113.csv

이후 실행

  # trend_data_organizer
